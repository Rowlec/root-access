## Proposal Output Rubric

Score each item from 1 to 5.

| Criterion | 1 | 3 | 5 | Score |
| --- | --- | --- | --- | --- |
| Problem clarity | Vague problem | Problem is understandable | Specific user, situation, pain, and importance |  |
| Target customer | Generic users | Basic segment | Reachable segment with behavior and need |  |
| Research direction | No verification path | Some competitor/trend notes | Clear competitors, sources, and unknowns to verify |  |
| Value proposition | Feature list only | Basic benefit | Clear difference and reason to care |  |
| Business model | Unrealistic or missing | Revenue idea exists | Payer, pricing logic, costs, and assumptions are clear |  |
| MVP scope | Too many features | Basic MVP list | Must-have/excluded features and test plan are clear |  |
| Pitch/proposal structure | Disorganized | Has sections | Clear flow from problem to solution to business case |  |
| Academic integrity | Copy-paste output | Some student editing | Output clearly needs review, verification, and student voice |  |
